
*yii/jui/Draggable*   >> |yii/jui/Widget|
===================

DESCRIPTION
------------------------------------------------------------------------------

 For example:

   Draggable::begin([
       'clientOptions' => ['grid' => [50, 20]],
   ]);

   echo 'Draggable contents here...';

   Draggable::end();
