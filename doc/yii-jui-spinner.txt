
*yii/jui/Spinner*   >> |yii/jui/InputWidget|
=================

DESCRIPTION
------------------------------------------------------------------------------

 For example:

   echo ProgressBar::widget([
       'clientOptions' => [
           'value' => 75,
       ],
   ]);

 The following example will use the name property instead:

   php
   ProgressBar::begin([
       'clientOptions' => ['value' => 75],
   ]);

   echo '<div class="progress-label">Loading...</div>';

   ProgressBar::end();
