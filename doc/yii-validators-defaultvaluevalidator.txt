
*yii/validators/DefaultValueValidator*   >> |yii/validators/Validator|
======================================

DESCRIPTION
------------------------------------------------------------------------------

 DefaultValueValidator is not really a validator. It is provided mainly to
 allow specifying attribute default values when they are empty.
