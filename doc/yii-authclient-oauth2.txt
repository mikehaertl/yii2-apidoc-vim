DESCRIPTION
------------------------------------------------------------------------------

 In oder to acquire access token perform following sequence:

   'components' => [
       'authClientCollection' => [
           'class' => 'yii\authclient\Collection',
           'clients' => [
               'yandex' => [
                   'class' => 'yii\authclient\clients\YandexOpenId'
               ],
           ],
       ]
       ...
   ]
