
*yii/bootstrap/Html*   >> |yii/bootstrap/BaseHtml|
====================

DESCRIPTION
------------------------------------------------------------------------------

 This class inherits all functionality available at '\yii\helpers\Html' and
 can be used as substitute.

 Attention: do not confuse '\yii\bootstrap\Html' and '\yii\helpers\Html', be
 careful in which class you are using inside your views.
