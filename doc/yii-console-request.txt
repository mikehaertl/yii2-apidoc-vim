
*yii/console/Request*   >> |yii/base/Request|
=====================

DESCRIPTION
------------------------------------------------------------------------------

 It is a wrapper for the PHP '$_SERVER' variable which holds information about
 the currently running PHP script and the command line arguments given to it.
