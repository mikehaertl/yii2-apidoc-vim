
*yii/jui/Resizable*   >> |yii/jui/Widget|
===================

DESCRIPTION
------------------------------------------------------------------------------

 For example:

   Resizable::begin([
       'clientOptions' => [
           'grid' => [20, 10],
       ],
   ]);

   echo 'Resizable contents here...';

   Resizable::end();
