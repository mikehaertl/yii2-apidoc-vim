
*yii/bootstrap/ButtonDropdown*   >> |yii/bootstrap/Widget|
==============================

DESCRIPTION
------------------------------------------------------------------------------

 For example,

   // a button group using Dropdown widget
   echo ButtonDropdown::widget([
       'label' => 'Action',
       'dropdown' => [
           'items' => [
               ['label' => 'DropdownA', 'url' => '/'],
               ['label' => 'DropdownB', 'url' => '#'],
           ],
       ],
   ]);
