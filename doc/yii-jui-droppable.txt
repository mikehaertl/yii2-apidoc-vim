
*yii/jui/Droppable*   >> |yii/jui/Widget|
===================

DESCRIPTION
------------------------------------------------------------------------------

 For example:

   Droppable::begin([
       'clientOptions' => ['accept' => '.special'],
   ]);

   echo 'Droppable body here...';

   Droppable::end();
