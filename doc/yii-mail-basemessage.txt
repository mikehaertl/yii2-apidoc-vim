
*yii/mail/BaseMessage*   >> |yii/base/Object|
======================

SUBCLASSES
------------------------------------------------------------------------------

|yii/swiftmailer/Message|

DESCRIPTION
------------------------------------------------------------------------------

 By default, 'send()' will use the "mail" application component to send the
 current message. The "mail" application component should be a mailer instance
 implementing 'MailerInterface'.


PROPERTY DETAILS
------------------------------------------------------------------------------

 (\yii\mail\MailerInterface) The mailer instance that created this message.
 For independently created messages this is 'null'.
