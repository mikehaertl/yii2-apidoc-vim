
*yii/console/controllers/ServeController*   >> |yii/console/Controller|
=========================================

DESCRIPTION
------------------------------------------------------------------------------

 In order to access server from remote machines use 0.0.0.0:8000. That is
 especially useful when running server in a virtual machine.
