
*yii/bootstrap/Collapse*   >> |yii/bootstrap/Widget|
========================

DESCRIPTION
------------------------------------------------------------------------------

 For example:

   echo Collapse::widget([
       'items' => [
           // equivalent to the above
           'Collapsible Group Item #1' => [
               'content' => 'Anim pariatur cliche...',
               // open its content by default
               'contentOptions' => ['class' => 'in']
           ],
           // another group item
           'Collapsible Group Item #2' => [
               'content' => 'Anim pariatur cliche...',
               'contentOptions' => [...],
               'options' => [...],
           ],
       ]
   ]);

$items					*yii/bootstrap/Collapse::$items*
 (array) List of groups in the collapse widget. Each array element represents
 a single group with the following structure:

   // item key is the actual group header
   'Collapsible Group Item #1' => [
       // required, the content (HTML) of the group
       'content' => 'Anim pariatur cliche...',
       // optional the HTML attributes of the content group
       'contentOptions' => [],
       // optional the HTML attributes of the group
       'options' => [],
   ]
