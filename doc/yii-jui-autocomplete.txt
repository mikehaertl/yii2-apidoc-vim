
*yii/jui/AutoComplete*   >> |yii/jui/InputWidget|
======================

DESCRIPTION
------------------------------------------------------------------------------

 For example:

   echo Selectable::widget([
       'items' => [
           'Item 1',
           [
               'content' => 'Item2',
           ],
           [
               'content' => 'Item3',
               'options' => [
                   'tag' => 'li',
               ],
           ],
       ),
       'options' => [
           'tag' => 'ul',
       ],
       'itemOptions' => [
           'tag' => 'li',
       ],
       'clientOptions' => [
           'tolerance' => 'fit',
       ],
   ]);

 The following example will use the name property instead:

   [
       'content' => 'item content',
       // the HTML attributes of the item container tag. This will overwrite
 "itemOptions".
       'options' => [],
   ]
