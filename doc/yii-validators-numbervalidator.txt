
*yii/validators/NumberValidator*   >> |yii/validators/Validator|
================================

DESCRIPTION
------------------------------------------------------------------------------

 The format of the number must match the regular expression specified in
 'integerPattern' or 'numberPattern'. Optionally, you may configure the 'max'
 and 'min' properties to ensure the number is within certain range.

$integerOnly				*yii/validators/NumberValidator::$integerOnly*

 (bool) Whether the attribute value can only be an integer. Defaults to false.

$max					*yii/validators/NumberValidator::$max*

 (int|float) Upper limit of the number. Defaults to null, meaning no upper
 limit.

$min					*yii/validators/NumberValidator::$min*

 (int|float) Lower limit of the number. Defaults to null, meaning no lower
 limit.

$tooBig					*yii/validators/NumberValidator::$tooBig*

 (string) User-defined error message used when the value is bigger than 'max'.

$tooSmall				*yii/validators/NumberValidator::$tooSmall*

 (string) User-defined error message used when the value is smaller than
 'min'.

$integerPattern				*yii/validators/NumberValidator::$integerPattern*

 (string) The regular expression for matching integers.

$numberPattern				*yii/validators/NumberValidator::$numberPattern*

 (string) The regular expression for matching numbers. It defaults to a
 pattern that matches floating numbers with optional exponential part (e.g.
 -1.23e-10).
